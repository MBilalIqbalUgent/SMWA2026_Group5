## Overview  
This project analyses how social media discussions and web data can be used to understand and predict movie performance. The work is divided into descriptive analysis and predictive modelling.

---

## Data  
- IMDb data (ratings, budget, box office)  
- Reddit discussions and reviews  
- Processed dataset: `movie_data_professional.csv`  

---

## Methodology  

### Descriptive  
- Text cleaning and preprocessing  
- Sentiment analysis  
- Trend and keyword analysis  

### Predictive  
- Feature engineering from text and sentiment  
- Model training and evaluation  
- Performance prediction using social media signals  

---

## Key Insight  
Social media sentiment and engagement provide useful signals for understanding and partially predicting movie performance.
